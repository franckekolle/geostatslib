# geostatlib 

**geostatlib** est une bibliothèque Python dédiée à la modélisation géostatistique, 
l'analyse spatiale et l'interpolation de données géoscientifiques. Il est basé sur gstlearn. 

Conçue pour les débutants et chercheurs en géosciences, elle permet de simplifier les processus danalyse 
et de transformation des jeux de données bruts en modèles spatiaux, en prenant en charge aussi bien les 
variables continues que catégorielles.

## Fonctionnalités Clés

* **Analyse de Variographie** : Calcul et modélisation de variogrammes expérimentaux.
* **Interpolation Avancée** : Krigeage simple, ordinaire et universel.
* **Variables Mixtes** : Support pour la cosimulation de variables continues et lithologiques (catégorielles).
* **Variables catégorielles** : Support pour les simulations plurigausssiennes (catégorielles).
* **Traitement de Données** : Filtrage de bruit (Gaussian noise) et décomposition de signaux.
* **Analyse Multivariée** : Intégration de l'Analyse en Composantes Principales (ACP) pour les données géochimiques.

## Installation

Vous pouvez installer la version de développement via pip :
